# Getting Started With MyMiix Chatrooms

This project is a mymiix chatroom.<br />

## Features

- ✅ Create Chatrooms.
- ✅ Dropin & talk.
- ✅ Tap users Profile & Tap mute.
- ✅ Become Admin or Moderator.
- ✅ Server Mute user.
- ✅ Ban user.
- ✅ Admins can assign moderators.
- ✅ Password protect Rooms.

## Technology used

- Using NodeJS
- Using PeerJS
- Using ReactJS
